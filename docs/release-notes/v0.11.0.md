# v0.11.0 Release Notes

**Release Date:** 2026-03-28

## Overview

v0.11.0 delivers cross-cloud drift correlation, provider status monitoring, and unified Grafana dashboards with Azure support.

## Changes

### Cross-Cloud Drift Correlation Engine (#133)

- Correlation engine links related drifts across AWS, GCP, and Azure
- Unified dashboard for multi-cloud drift visibility
- Temporal and resource-based correlation rules

### Provider Status UI & Grafana Dashboard (#134)

- Provider Status API with health monitoring per cloud provider
- Grafana dashboards extended with Azure panels
- Real-time provider connectivity status in Dashboard UI

### Azure Activity Log Integration (#132)

- Azure Activity Log parser integrated into Falco subscriber
- E2E test coverage for Azure drift detection pipeline
- Full three-cloud (AWS + GCP + Azure) event processing

Closes **v0.11.0** milestone (3/3 issues).
